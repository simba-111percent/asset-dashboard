구글시트 연동
HW 자산 · SW 자산 · 좌석배치도 시트를 각각 연동해요. 연동 후 5분마다 자동 갱신됩니다.
1. Apps Script 웹앱 URL 설정

연동 URL이 고정 설정되어 있어요. 별도 입력 없이 접속하면 자동으로 연동됩니다.

2. 최신 데이터 가져오기

페이지 접속 시 자동으로 가져오며, 5분마다 갱신됩니다. 즉시 갱신하려면 아래 버튼을 클릭하세요.

Apps Script 코드 (3개 시트 모두 동일)

각 구글시트 열기 → 확장 프로그램 → Apps Script → 기존 코드 전체 삭제 후 아래 코드 붙여넣기 → 배포 → 새 배포 → 웹앱 → 액세스: 전체 허용 → URL 복사 후 위에 입력
